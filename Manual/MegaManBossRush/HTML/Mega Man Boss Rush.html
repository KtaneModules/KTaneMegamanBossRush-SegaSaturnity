Keep Talking and Nobody Explodes Mod Mega Man Boss Rush

On the Subject of Mega Man Boss Rush

This one is Hard Man.

On the module there is a 3 by 3 grid of frames containing Mega Man in the center and 8 Robot Masters surrounding him.

Find the correct sequence of Robot Masters based on each boss' weaknesses. Every boss selected must be weak to the last Robot Master selected's weapon (with the exception to the first boss).

First boss

Calculate the number of batteries plus the number of ports on the bomb, and then multiply this value by the last digit of the serial number. Then, add 1 to that number. If your number is higher than 8, subtract 8 until that number is between 1-8. Select the character in that position, reading order (ignoring Mega Man in the center), for the first selection.
